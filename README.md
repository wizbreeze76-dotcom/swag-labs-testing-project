# Swag Labs电商网站全栈测试项目

**项目亮点**（可直接复制到简历）  
- 独立设计并执行 **33个UI测试用例** + **6个接口测试用例**（覆盖功能、性能、兼容性、安全、6个预设账号）
- 使用 **Playwright + POM** 实现核心UI自动化（覆盖率95%）
- 使用 **Apifox** 对 JSONPlaceholder 进行接口CRUD测试 + 场景用例串联
- 使用 Chrome DevTools Performance 面板完成加载时间对比测试
- 发现并通过 **GitHub Issues** 管理 **2个真实Bug**
- 技术栈：PyCharm + Playwright + Pytest + Apifox + GitHub Issues

## 项目结构

swag-labs-testing-project/
├── automation/          # Playwright UI自动化脚本
├── test-cases/          # UI测试用例执行表
├── api-tests/           # Apifox接口测试集合
├── docs/                # 测试计划 + 总结报告
├── reports/             # HTML报告
├── bugs/                # Bug截图
└── requirements.txt
